# PRODIGY_GA_02

# Image Generation with Pre-trained Models

## 📌 Objective

Develop a Python application that generates images from text prompts using a pre-trained image generation model such as Stable Diffusion.

---

## 📖 Project Description

This project demonstrates how to generate AI-created images from natural language prompts using the Stable Diffusion v1.5 pre-trained model provided by Hugging Face.

The application accepts a text prompt from the user, loads the pre-trained model, generates an image, and automatically saves it inside the `generated_images` folder.

---

## 🚀 Features

- Text-to-Image Generation
- Uses Stable Diffusion v1.5
- Interactive Prompt Input
- Automatically Saves Generated Images
- Supports Multiple Prompt Generations
- Simple Command-Line Interface

---

## 🛠️ Technologies Used

- Python 3.11
- Stable Diffusion v1.5
- Hugging Face Diffusers
- Transformers
- PyTorch
- Pillow
