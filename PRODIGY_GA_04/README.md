# PRODIGY_GA_04
# Image-to-Image Translation using Pre-trained Generative Models

## 📌 Overview

This project is developed as part of the **Prodigy InfoTech Generative AI Internship**.

The objective is to perform **Image-to-Image Translation** by using a pre-trained generative AI model. The application accepts an input image and a text prompt describing the desired transformation, then generates a new image accordingly.

---

## 🚀 Features

- Load an input image
- Accept user-defined transformation prompts
- Generate new images using a pre-trained AI model
- Save generated images automatically
- Easy command-line interface

---

## 🛠 Technologies Used

- Python
- PyTorch
- Hugging Face Diffusers
- Transformers
- Stable Diffusion
- Pillow

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/PRODIGY_GA_04.git
```

Navigate to the project

```bash
cd PRODIGY_GA_04
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Project

```bash
python image_translation.py
```

Enter a prompt such as:

```
Anime artwork
```

or

```
Oil painting
```

or

```
Cyberpunk city
```

The generated image will be saved inside:

```
output_images/
```
