# 🤖 PRODIGY_GA_02 - Custom GPT-2 Chatbot

## 📌 Task

Develop a custom GPT-2 chatbot by fine-tuning a pre-trained GPT-2 model on a custom dataset using Hugging Face Transformers.

---

# Project Overview

This project demonstrates how to build and fine-tune a GPT-2 model for conversational text generation.

The chatbot is capable of generating AI responses based on prompts provided by the user through a terminal interface.

---

- Dataset Preparation
- GPT-2 Fine-Tuning

# Installation

```bash
git clone https://github.com/yourusername/PRODIGY_GA_02.git

cd PRODIGY_GA_02

python -m venv genai_env

genai_env\Scripts\activate

pip install -r requirements.txt
```

# Output

The chatbot accepts user input and generates responses using the fine-tuned GPT-2 model.

Example:

```
You: Hello

GPT-2:
Hello! How can I assist you today?

You: Tell me about AI

GPT-2:
Artificial Intelligence enables machines to learn from data and perform intelligent tasks.
```
