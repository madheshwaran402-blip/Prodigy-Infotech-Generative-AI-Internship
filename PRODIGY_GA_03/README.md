# PRODIGY_GA_03
## Text Generation with Markov Chains

### 📌 Task Description

This project was completed as part of the **Prodigy InfoTech Generative AI Internship**.

The objective is to build a simple text generation application using **Markov Chains**, which predicts the probability of the next word based on previously observed words.

---

## Features

- Generates AI-related text using Markov Chains
- Uses a custom training corpus
- Interactive command-line interface
- Generates different sentences on each execution
- Beginner-friendly implementation

---

## Technologies Used

- Python 3.x
- Markovify
- VS Code

## Installation

Clone the repository

```bash
git clone https://github.com/your-username/PRODIGY_GA_03.git
```

Move into the project

```bash
cd PRODIGY_GA_03
```

Create Virtual Environment

```bash
python -m venv genai_env
```

Activate Environment

Windows

```bash
genai_env\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## Run the Project

```bash
python text_generator.py
```
